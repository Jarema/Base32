# Base32 for Swift on OS X and iOS
[![MIT License](https://img.shields.io/badge/license-MIT-blue.svg?style=flat)](LICENSE)

Base32 is a [Base32](https://tools.ietf.org/html/rfc4648) implementation for Swift.

## Requirements

* iOS 9 or later
* macOS 10.9 or later
* tvOS 9 or later
* watchOS 2 or later
